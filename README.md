# soal-tes-psikotest (Next.js)

Aplikasi latihan psikotes berbasis Next.js (siap deploy ke Vercel) dengan penilaian otomatis per materi:

- Sinonim
- Antonim
- Analogi
- Deret Angka/Huruf

## Menjalankan lokal

```bash
npm install
npm run dev
```

Buka http://localhost:3000

## Build produksi

```bash
npm run build
npm run start
```
